Updates

### (January 2026) New Service launched: NVDB TN-ITS Export API The Norwegian Public Road Administration (NPRA) has launched the NVDB TN-ITS Export API, a public service that provides continuous updates of road object changes from the National Road Database (NVDB). **The service delivers daily incremental updates for four key datasets:** - speed limits - road names - road numbers - height restrictions This launch is an important step in Norway's implementation of the EU ITS Directive and the Real-Time Traffic Information Regulation (EU 2022/670). The service follows the European TN-ITS standard (CEN/TS 17268), ensuring that road and traffic data can be shared in a harmonized, reliable, and standardized manner across Europe. **The API offers:** - Daily updates and monthly snapshots - A map viewer that visualizes road objects - XML data in TN-ITS format via an open REST API, with no login required The service is open to everyone: [NVDB TN-ITS Export API](https://nvdb-tnits.atlas.vegvesen.no/) Contact us: [tn-its@vegvesen.no](mailto:tn-its@vegvesen.no)
