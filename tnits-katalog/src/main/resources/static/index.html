NVDB TN-ITS Export API

This service provides standardized exports of Norwegian road network data from NVDB (Nasjonal vegdatabank) in TN-ITS format for European data exchange.

Access complete snapshots and incremental daily updates through our REST API, explore available datasets in the browser, verify data quality with the interactive map viewer.

Use Cases

For Service Providers

Set up automated integration with our REST API to consume TN-ITS data:

  • Browse available datasets in the dataset browser
  • Download monthly snapshots or daily incremental updates
  • GML 3.2 format following TN-ITS specification

View API Documentation →

For Data Verification

Verify OpenLR location references and validate data quality:

  • Browse and download datasets from the dataset browser
  • Visualize road network and features on interactive map
  • Inspect OpenLR encoding and geometry accuracy
